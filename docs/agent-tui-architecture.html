agent-tui Architecture

Terminal UI scaffold for AI coding agents · Python 3.11+ · Textual + DeepAgents + LangGraph

UI Layer — Textual 8.x

  • • app.py — main orchestration (2,000+ lines)
  • • ChatInput + autocomplete for slash commands
  • • Virtualized MessageStore (1,000+ message histories)
  • • HITL: tool approval + ask_user prompts
  • • StatusBar: live token count + % context used
  • • 15+ widget types (model, theme, thread selectors)

Agent Backends

  • • AgentProtocol — pluggable backend contract
  • • StubAgent — scripted mock for development
  • • DeepAgentsAdapter — wraps LangGraph agent
  • • EventTranslator — LangGraph → AgentEvent stream
  • • LocalShellBackend — file + shell ops in cwd
  • • Memory (AGENTS.md) + Skills (.md files)

External Integrations

  • • OpenAI API — gpt-4o / gpt-5.2 via langchain-openai
  • • Tavily API — web search (TAVILY_API_KEY)
  • • httpx + markdownify — URL fetch → Markdown
  • • MCP — Model Context Protocol tool servers
  • • SQLite (aiosqlite) — thread + checkpoint persistence
  • • LangGraph InMemoryStore — cross-thread state
